# 🌳 MongoDB Atlas 决策树

## 根据你的截图，回答以下问题：

### 问题1：是否已登录？
- **否** → 点击 "Sign in with GitHub"
- **是** → 继续问题2

### 问题2：是否看到套餐选择？
- **是** → 滚动到底部，点击 "M0 Sandbox (FREE)" 的 "Create"
- **否** → 继续问题3

### 问题3：是否看到集群配置？
- **是** → 
  1. Cloud Provider: AWS
  2. Region: Singapore 或 Tokyo
  3. Cluster Name: pan-gu-ai-cluster
  4. 点击 "Create Cluster"
- **否** → 继续问题4

### 问题4：是否看到创建用户？
- **是** →
  1. Username: pan-gu-ai-user
  2. Password: 设置密码
  3. 权限: Read and write to any database
  4. 点击 "Add User"
- **否** → 继续问题5

### 问题5：是否看到网络配置？
- **是** →
  1. 选择 "Allow Access from Anywhere"
  2. IP: 0.0.0.0/0
  3. 点击 "Confirm"
- **否** → 继续问题6

### 问题6：是否看到连接选项？
- **是** →
  1. 选择 "Connect your application"
  2. Driver: Node.js
  3. 点击 "Copy"
- **否** → 描述当前界面

## 🚨 紧急情况

### 如果看到错误：
- **"Invalid credentials"** → 检查用户名密码
- **"Network access not configured"** → 配置0.0.0.0/0
- **"Cluster creation failed"** → 换区域重试
- **其他错误** → 告诉我错误信息

### 如果找不到选项：
- 检查左侧菜单
- 使用页面搜索
- 刷新页面
- 换浏览器

## 🎯 快速通道

### 如果你看到：
- **"Build a Database"** → 点击它
- **"M0 Sandbox"** → 点击 "Create"
- **"Configure Cloud Provider"** → 配置后点击 "Create Cluster"
- **"Add New Database User"** → 填写信息点击 "Add User"
- **"Add IP Address"** → 选择0.0.0.0/0点击 "Confirm"
- **"Connect"** → 点击它，然后选择 "Connect your application"
